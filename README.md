# Dashboard Meteorológico

Dashboard em tempo real com dados de temperatura, umidade e vento de cidades brasileiras.

## Tecnologias

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Recharts
- API Open-Meteo (gratuita, sem token)

## Como rodar localmente

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000)

## Funcionalidades

- Temperatura, umidade e vento em tempo real
- Cores dinâmicas por faixa de temperatura
- Gráfico de histórico das últimas 24h
- Seletor de cidade
- Interface responsiva
